ComposeKey by Jan Ouwens

Brings the power of Linux's Compose Key to macOS through Alfred

What?

ComposeKey is an Alfred snippet bundle that simulates Linux's compose key.

Why?

I wrote a blog post about this which you can read here.

TL;DR: Entering special characters is a nuisance, even on macOS. To add an acute to a letter, you need ⎇e; to add an umlaut you need ⎇u. For œ, you somehow need ⎇q This is pretty hard to remember.

With ComposeKey, it's a lot easier. To get é, you simply type §'e. Ä is §"A and œ is §oe. Easy!
